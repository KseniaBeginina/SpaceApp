# 🌌 SpaceApp - NASA APOD Explorer

Android application that integrates NASA APOD (Astronomy Picture of the Day) API to display daily space images and detailed descriptions.


## 🚀 About the Project

SpaceApp is an Android application built with a focus on clean architecture and modern Android development practices.

The project implements:

- Fetching data from NASA APOD API
- Displaying a list of space images
- Detailed screen with full description
- Navigation between screens
- Network state handling
- MVVM architecture setup
